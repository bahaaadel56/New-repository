# 🌐 SuperMarket POS - نسخة الويب

## 📱 للتشغيل على الهاتف (بدون لابتوب!)

### الطريقة 1: رفع على استضافة مجانية (أسرع طريقة)

#### Netlify (الأسهل)
1. افتح الهاتف وادخل على: **netlify.com**
2. سجل بحساب Google (دقيقة واحدة)
3. اسحب ملفات هذه المجلد (الـ 4 ملفات) للمربع
4. يعطيك رابط مباشر مثل: `https://supermarket-pos-123.netlify.app`
5. افتح الرابط من أي هاتف!

#### GitHub Pages (مجاني للأبد)
1. سجل في github.com من الهاتف
2. أنشئ repo جديد
3. ارفع الملفات الأربعة
4. فعل GitHub Pages من الإعدادات

### الطريقة 2: تشغيل محلي (بدون إنترنت)

#### على Android:
1. حمل تطبيق **"Termux"** من Play Store
2. افتح Termux واكتب:
```bash
pkg install python
python -m http.server 8080
```
3. افتح المتصفح وادخل: `http://localhost:8080`

#### على iPhone:
1. حمل **"Pythonista 3"** من App Store (مدفوع)
2. أو استخدم **"iSH"** (مجاني)

### الطريقة 3: نقل الملفات للهاتف

#### عبر USB:
1. وصل الهاتف بالكمبيوتر (لو في كمبيوتر ثاني في البيت)
2. انقل المجلد للهاتف
3. افتح `index.html` من متصفح الهاتف

#### عبر Bluetooth / ShareIt:
1. أرسل المجلد من أي جهاز للهاتف
2. افتح `index.html`
